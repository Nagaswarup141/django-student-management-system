# StudentMS — Frontend

## How to Run
1. Make sure Django_API server is running: `python manage.py runserver`
2. Open `index.html` in browser (double click or Live Server)
3. Login: `admin` / `admin123`

## API Connection
- Django API: `http://127.0.0.1:8000/api`
- Change `API` variable in `index.html` if different port
